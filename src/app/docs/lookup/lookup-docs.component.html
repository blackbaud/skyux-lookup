The lookup component lets users search and make selections from long lists of options.

Use the lookup component when users make a single selection or multiple selections from a long list of options. For guidance on which selection inputs to use in other scenarios, see the form design guidelines.

Use a single-select lookup when users need to choose a single option, and use a multi-select lookup when users may choose one or more.

Include the button to create new options when users may need to add a new option to the list as part of completing their tasks. Clicking this button will begin whatever experience is necessary for creating the new option. When users complete that workflow, the new option is selected in the lookup field.

Use a custom results template when users need more information about the available options to make the correct choice. The custom format will appear in the popover match suggestions.
